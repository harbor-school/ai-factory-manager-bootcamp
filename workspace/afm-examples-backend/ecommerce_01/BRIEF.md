# Brief: LUEUR (뤼에르) — 여성 화장품 쇼핑몰

> 29CM 감성의 한국형 여성 화장품 단일 브랜드 쇼핑몰

## Requirements
- [x] 1. 회원가입/로그인 (아이디+비밀번호, 카카오 로그인)
- [ ] 2. 상품 목록 — 카테고리별 필터 (스킨케어, 메이크업, 바디/헤어, 향수, 뷰티소품)
- [ ] 3. 상품 상세 — 큰 이미지, 용량/쉐이드 옵션 선택, 가격, 상세 설명
- [ ] 4. 장바구니 — 상품 추가/수량 변경/삭제, 쿠폰 적용
- [ ] 5. 쿠폰 — % 할인 / 정액 할인 두 타입, 쿠폰 코드 입력 방식
- [ ] 6. 주문 — 배송지 입력 → 토스페이먼츠 결제 위젯 (테스트 키) → 주문 완료
- [ ] 7. 주문 상태 — 주문확인 → 배송준비 → 배송중 → 배송완료 (마이페이지에서 확인)
- [ ] 8. 리뷰 — 별점(1-5) + 텍스트 + 사진 첨부 + 피부타입 태그(건성/지성/복합성/민감성)
- [ ] 9. 마이페이지 — 주문 내역, 내 리뷰, 프로필 수정
- [ ] 10. 더미 상품 데이터 시드 API — 샘플 10~15개 자동 생성

## Constraints
- Express + PostgreSQL + 단일 index.html + Tailwind CDN (dangun_01 패턴)
- Vercel 배포
- 토스페이먼츠 테스트 키 사용
- 카카오 REST API 키는 환경변수로 관리

## Non-goals
- 관리자 페이지 (DB 직접 관리, 추후 별도 추가)
- 실결제 연동 (테스트 모드만)
- 재고 관리 시스템
- 배송 추적 연동

## Style
- 29CM 레퍼런스: 미니멀, 여백 많음, 큰 상품 이미지
- 블랙 & 화이트 톤 + 포인트 컬러 최소 사용
- 세리프 포인트 폰트
- 모바일 퍼스트 (max-w-md 중앙 정렬)

## Key Concepts
- **LUEUR**: 브랜드명, 프랑스어로 "광채"
- **피부타입 태그**: 리뷰에 건성/지성/복합성/민감성 태그를 달아 다른 사용자 참고
- **쉐이드**: 메이크업 상품의 색상 변형 옵션

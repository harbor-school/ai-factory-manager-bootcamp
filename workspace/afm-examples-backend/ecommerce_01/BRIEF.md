# Brief: 주방용품 쇼핑몰 (ecommerce_01)

> 요리 애호가 타깃 주방용품 쇼핑몰 — 상품 탐색부터 토스페이먼츠 결제까지 풀 플로우

## Requirements
- [x] 1. 이메일+비밀번호 회원가입/로그인 (JWT 인증)
- [x] 2. 상품 목록 페이지 — 카테고리 필터(칼/도마류, 냄비/팬류, 주방소품류), 15개 상품 표시
- [x] 3. 상품 상세 페이지 — 이미지, 이름, 가격(원화), 설명, 장바구니 담기
- [x] 4. 장바구니 — DB 저장, 수량 변경/삭제, 합계 표시
- [x] 5. 주문서 — 배송지 입력 후 토스페이먼츠 결제 위젯 연동
- [x] 6. 토스페이먼츠 테스트 키로 실제 결제 플로우 동작
- [x] 7. 주문 완료 및 주문 내역 조회
- [x] 8. 시드 데이터: 3카테고리 x 5상품 = 15개 (가격대 1만~15만원)
- [x] 9. 상품 이미지: fal-image-gen 생성 → Supabase Storage 업로드
- [x] 10. Vercel 배포 후 접근 가능한 URL 확보

## Constraints
- Express + PostgreSQL(Supabase) 백엔드, 단일 server.js
- React + Tailwind CDN 프론트, 단일 index.html
- 기존 todo_app_01과 동일한 Supabase 인스턴스
- 토스페이먼츠 테스트 환경

## Non-goals
- 관리자 페이지 없음 (상품은 시드 데이터로 관리)
- 실 결제 환경 (프로덕션 키) 연동 아님
- 재고 차감 로직, 배송 추적, 리뷰 시스템 없음
- SEO, 커스텀 도메인 연결 없음

## Style
- 따뜻한 우드톤/베이지 컬러 베이스
- 요리 취미 감성 — 깔끔하고 정돈된 느낌
- 모바일 반응형

## Key Concepts
- **시드 데이터**: DB 초기화 시 자동 삽입되는 상품 목록
- **토스페이먼츠 위젯**: 결제 UI를 iframe으로 임베드하는 방식
- **Supabase Storage**: 상품 이미지 파일 호스팅

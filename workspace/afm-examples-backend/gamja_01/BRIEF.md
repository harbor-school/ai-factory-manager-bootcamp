# Brief: 당근마켓 클론 (gamja_01)

> 중고거래 핵심 기능을 완전 재현한 당근마켓 클론 — 위치 기반 상품 탐색, 1:1 채팅, 매너온도 시스템 포함.

## Requirements
- [x] 1. 이메일/비밀번호 회원가입 및 로그인
- [x] 2. 상품 CRUD (제목, 설명, 가격, 카테고리, 이미지 최대 10장, 거래 장소)
- [x] 3. 거래 상태 관리: 판매중 / 예약중 / 거래완료
- [x] 4. 가격 입력 + "가격 제안 받기" 옵션 + 나눔(무료) 옵션
- [x] 5. 브라우저 Geolocation 기반 동네 설정, 반경(3km/5km/10km) 필터링
- [x] 6. 상품 검색 (제목+내용 텍스트) + 카테고리 필터
- [x] 7. 1:1 채팅 (3초 polling 방식) + 상품 찜(좋아요) 기능
- [ ] 8. 매너온도 시스템: 36.5도 시작, 거래 후기에 따라 ±0.1~0.5도 변동
- [ ] 9. 거래 후기 작성 (거래완료 후)
- [x] 10. 이미지 업로드 (Supabase Storage)

## Constraints
- 프론트엔드: single index.html (React CDN + Tailwind CSS)
- 백엔드: single server.js (Express)
- DB: PostgreSQL (Supabase) — 테이블 prefix `gamja_01_`
- 테이블: users, products, chats, messages, reviews
- 카테고리: 하드코딩 10개 (디지털기기, 가구, 의류 등)
- 이미지 업로드: supabase-image-uploader 에이전트 사용

## Non-goals
- 동네생활 (커뮤니티) 기능
- 실시간 WebSocket 채팅
- 소셜 로그인 / 전화번호 인증
- 프로덕션 배포 (로컬 개발 기준)

## Style
- 당근마켓 UI/UX 최대한 유사하게 — 주황색 메인 컬러, 카드형 상품 리스트
- 모바일 퍼스트 레이아웃
- 직관적이고 깔끔한 인터페이스

## Key Concepts
- **매너온도**: 사용자 신뢰도 지표. 36.5도 기본, 거래 후기 평가로 변동
- **동네 인증**: GPS 좌표 기반으로 현재 위치 동네 설정
- **나눔**: 가격 0원, 무료 나눔 거래
- **거래 상태**: 판매중 → 예약중 → 거래완료 흐름
